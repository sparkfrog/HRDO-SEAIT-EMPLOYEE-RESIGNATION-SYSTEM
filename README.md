## Overview

This system is designed to **record, manage, and track data of employees who have resigned** from the institute. It allows HR personnel to efficiently handle resignation records, including employee details, separation dates, reasons for resignation, and other relevant information.  

---

## Features

- Record details of resigned employees  
- Manage resignation data by department and year  
- Track changes and updates made to employee records  
- User access limited to HRDO staff  

---

## Notes

This system is **strictly for internal use by SEAIT HRDO** and should not be shared or accessed by unauthorized personnel.
