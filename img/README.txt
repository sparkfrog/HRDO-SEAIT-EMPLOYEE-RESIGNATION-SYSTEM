IMAGES FOLDER
==============

This folder contains images for the HRDO Employee Resignation System.

REQUIRED IMAGES:
----------------

1. seaitLogo.png
   - SEAIT logo for favicon and login page
   - Recommended size: 140x140 pixels or larger (square)
   - Format: PNG with transparent background (recommended)
   - Used in: Login page header and browser favicon

2. background.jpg
   - Background image for login page
   - Recommended size: 1920x1080 pixels or larger
   - Format: JPG or PNG
   - Used in: Login page background

NOTES:
------
- Place your seaitLogo.png and background.jpg files in this folder
- The system will not display properly without these images
- For best results, use high-quality images
- Ensure file names match exactly (case-sensitive on some systems)

FAVICON:
--------
The seaitLogo.png is used as the browser favicon (tab icon) for all pages:
- index.php (login page)
- All admin pages
- All HR staff pages
