## ğŸ” Secure Your System (2 minutes)

### 1. Change Admin Password
1. Login as admin
2. Click **Profile Settings**
3. Enter current password: `password`
4. Enter new strong password
5. Confirm new password
6. Click **Update Profile**

### 2. Add Departments
1. Click **Departments**
2. Click **Add New Department**
3. Enter department name
4. Click **Save Department**
5. Repeat for all departments

### 3. Update HR Staff Account
1. Go to **System Users**
2. Click **Edit** on HR Staff account
3. Update full name
4. Change password
5. Click **Update User**
